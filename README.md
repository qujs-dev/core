# Qu.js

Qu.js — это утилитарная JavaScript-библиотека для асинхронной, управляемой загрузки ассетов и связи модулей через события и Promise‑based API.

В основе лежит очередь `Que`, которая позволяет откладывать и упорядочивать выполнение кода до нужных моментов: готовность DOM, загрузка страницы, подключение внешних библиотек или другие кастомные события.

- Сайт проекта: [https://qujs.ru/](https://qujs.ru/)
- Документация: [https://qujs.ru/api/](https://qujs.ru/api/)
- Лицензия: MIT (см. файл `LICENSE`)

Qu.js родилась не из теории, а из практики. Из разных проектов постепенно вырисовался устойчивый набор задач: асинхронная загрузка, загрузка ассетов, связь модулей, ожидание готовности страницы и сторонних библиотек. Со временем этот служебный фундамент логично сложился в отдельную библиотеку, которую теперь можно просто подключать в новые проекты вместо того, чтобы собирать всё с нуля.

Это просто коллекция инструментов, которая когда-то оказалась удобной в моих проектах. Если вам близок такой подход, можете смело забирать и использовать у себя.
